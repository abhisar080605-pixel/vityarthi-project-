# Menstrual Cycle Predictor (Python Project)

## Project Overview
This is a simple Python project that predicts the **next menstrual cycle date** using:
- Last period start date  
- Average cycle length  

It also calculates an estimated **fertile window** based on common cycle patterns.

 **Disclaimer:**  
This tool is for **educational and practice purposes only**. It is **not a medical tool** and should not be used for health decisions.

---

##  Features
- Accepts user input for last period date.
- Predicts next expected period date.
- Estimates fertile window.
- Contains input validation for date correctness.
- Uses basic Python concepts:
  - Functions  
  - Condition checks  
  - Date and time handling  
  - User input  

---

##  Technologies Used
- **Python 3**
- `datetime` module
